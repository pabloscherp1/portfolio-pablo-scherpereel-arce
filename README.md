# Pablo Scherpereel Arce — Portfolio

Personal portfolio site. Built with React and Vite, deployed to GitHub Pages.

**Live:** https://pabloscherp1.github.io/portfolio-pablo-scherpereel-arce/

---

## Stack

- React 19 + Vite
- Framer Motion for animations
- Plain CSS (no framework)
- `gh-pages` for deployment

## Dev

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```

Builds and pushes to the `gh-pages` branch.
